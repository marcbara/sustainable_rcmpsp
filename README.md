# Trading Delay Penalties for Water Compliance

**Multi-Objective Scheduling of Construction Portfolios under Drought**

> Lostumbo, M. A., Bara Iniesta, M., & Calvet, L. (2026).
> *International Journal of Construction Management* — under review.

---

## What this repo does

This repository contains the full Python implementation of a **MOEA/D-based
bi-objective scheduler** for construction portfolio management under stochastic
water-quota restrictions.

The algorithm simultaneously minimises:
- **Z₁** — tardiness penalties (€) across all projects
- **Z₂** — daily water-quota violations (m³) under drought conditions

Water restriction sequences are generated by a two-state Markov chain calibrated
to three Mediterranean drought scenarios (mild / moderate / severe), handled via
Sample Average Approximation (SAA).

## Repository structure

```
sustainable_rcmpsp/
├── src/
│   ├── moead.py        # MOEA/D algorithm with water-arbitrage operator
│   ├── sgs.py          # Serial Schedule Generation Scheme decoder
│   ├── evaluator.py    # Bi-objective evaluator (Z1, Z2)
│   ├── scenario.py     # Markov-chain drought scenario generator
│   └── loader.py       # Portfolio JSON loader
├── portfolio.json       # Three-project construction portfolio (130 activities)
├── run_experiments.py   # Experiment runner (dev / paper modes)
├── run.py               # Quick single-run entry point
└── results/             # Output Pareto fronts and convergence plots
```

## Quickstart

```bash
pip install -r requirements.txt

# Single run (mild scenario, fast settings)
python run.py

# Full paper experiments (pop=100, gen=300, all three scenarios)
python run_experiments.py --mode paper
```

## Key results

| Scenario | \|PF\| | Water savings vs. baseline | Run time |
|----------|--------|---------------------------|----------|
| Mild     | 5      | up to **19.4%**           | ~11 min  |
| Moderate | 3      | up to **13.3%**           | ~8 min   |
| Severe   | 1      | **10.3%**                 | ~8 min   |

Under mild and moderate drought, MOEA/D also recovers **€36,500** in tardiness
penalties that the naive baseline incurs. The marginal cost of water compliance
is ~€14.85/m³ (mild scenario), directly readable from the Pareto front slope.
